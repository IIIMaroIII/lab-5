4.joinWords
Method signature:
public static String joinWords(String[] words)
Join words from words array and return as a string in the following format: "[str_1, str_2, ..., str_n]". If words is null or empty return null. words is guaranteed to not contain null values. words may contain empty strings, ignore them, i. e. don't put them in the resulting string. If words contains only empty strings return null.
